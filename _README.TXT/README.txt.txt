Py2Learn

Py2Learn é um Curso de Python online, desenvolvido através de Notebooks da Jupyter, que é uma plataforma livre 
que permite com que sejam criadas aplicações que permitem ao utilizador utilizar código e executá-lo em tempo real.

Instalação

Não há necessidade de instalar nada neste projeto, somente aceder ao link do curso, de seguida ler o conteúdo da 
Página do GitHub que dá uma breve instrução de como utilizar os notebooks e clicar no ícone do Binder que irá redirecioná-lo 
ao repositório dos Notebooks do curso. Quando entrar no repositório, é aconselhá-vel escolher o notebook "Main.ipynb" e 
começar a explorar tudo.
